O objetivo do nosso código foi facilitar para os clientes da John Deere a cuidar do solo 
de seus respectivos plantios com base em dados de clima, umidade e relevo do Brasil numa 
generalização das regiões. Nosso código foi espécifico para plantios de soja, 
pois quisemos testar o questionário e a tela de login feita para os usuários antes de colocar 
o conteúdo para assim, futuramente, caso o projeto continue, o código possuir uma base de dados 
mais ampla dentro de alguma Query com o SQL.
